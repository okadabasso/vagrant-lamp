# vagrant-lamp
to create centos 7.1 vm with vagrant and to install httpd, mariadb 10, php 7 on Windows10.

## Environment

* Windows10
* vagrant 1.8.1
* virtual box 5.0.16
* chefdk 0.12.0
* box bento/centos-7.1

## First Steps

* install vagrant, virtual box, chefdk
* do "vagrant init"
* do " berks vendor cookbooks" on chef-repo
* do "vagrant up"
